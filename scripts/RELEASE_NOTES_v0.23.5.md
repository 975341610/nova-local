# Nova v0.23.5 Release Notes

发布日期：2026-05-08
通道：stable
最低基线：v0.23.0

## 关键修复

- 修复笔记版本历史恢复后的历史列表刷新异常，避免关闭列表后再次打开变空。
- 修复版本恢复后笔记内容被错误写入元数据或 HTML 片段的问题，保护正文、双向链接和阅读模式渲染。
- 增强本地离线更新安装流程：安装失败会记录明确阶段、完整 traceback、目标版本、上一版本和回滚结果。
- 增强启动健康检查和自动回滚，避免升级失败后应用不可用。
- 远程更新下载增加包大小和 sha256 校验，防止损坏包进入安装阶段。

## 打包与发布

- 新增 `package_release.bat` / `一键打包更新包.bat` 一键命令行打包入口。
- 新增 `scripts/package_release.ps1`，自动执行前端构建、payload staging、运行时状态排除、签名打包、包内容复查和 `latest.json` 生成。
- release 模式会生成签名 `.nova-update`，可用于软件内 `设置 -> 更新`。
- dev 模式只生成未签名测试包，已签名客户端会拒绝安装。

## 失败排查

- 打包日志写入 `logs/package-release/`。
- 软件内安装失败日志写入 `data/logs/updater-install-failures.log`。
- 设置中的失败日志入口会合并显示安装失败日志和启动崩溃日志。
