# Nova v0.25.0 Release Notes

Release date: 2026-05-19
Channel: stable

## Changes

- Advanced table interactions refined: persistent row/column edge controls, cleaner dot-to-plus insert affordances, right-click toolbar behavior, multi-cell selection actions, and column-width resizing.
- Advanced table selection is less visually intrusive: selected cells now keep their original background and show only a thin selection border.
- Batch cell operations improved: clearing cells and applying cell background colors work across multi-cell selections.
- QingZhi UI iteration continued with updated editor/table styling and V9 visual refinements.
- Frontend package version updated to 0.25.0.

## Upgrade Notes

- Install this update package from Settings -> Update.
- Restart the app after installation.
- If the desktop app is already running, close and reopen it after the update so the refreshed frontend assets are loaded.
